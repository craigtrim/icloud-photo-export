iCloud Photo Export

Bulk download & backup iCloud Photos to your Mac

macOS Bash Free & Open Source

Quick Start

# Install dependency
pip install icloudpd

# Download photos from July 2025
./icloud_photo_export.sh 2025-07 -a your@email.com -o ~/Photos

# Convert HEIC to JPG
./convert_heic.sh -i ~/Photos/2025-07

What's Included

icloud_photo_export.sh
Download iCloud Photos by month, day, or date range
convert_heic.sh
Batch convert HEIC/HEIF to JPG using macOS sips

Features

Date-Based Downloads

Download by month (2025-07) or specific day (2025-07-15)

Range Support

Use --to for multi-month or multi-day ranges

HEIC to JPG

Convert iPhone photos to universal JPG format

Zero Dependencies

Uses macOS built-in tools + icloudpd

Examples

# Single month
./icloud_photo_export.sh 2025-07 -a you@email.com -o ~/Photos

# Date range (July - September)
./icloud_photo_export.sh 2025-07 --to 2025-09 -a you@email.com -o ~/Photos

# Include videos and Live Photos
./icloud_photo_export.sh 2025-07 -a you@email.com -o ~/Photos -v

# Convert HEIC, keep originals
./convert_heic.sh -i ~/Photos/2025-07 --keep-originals
